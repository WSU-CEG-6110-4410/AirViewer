void edu.wright.airviewer2.DocumentCommandWrapper.save ( String  aPath) throws IOException
inline

Save the PDF information including annotations encapsulated by the receiver to a file at the specified path.

The saved information contains the same annotation and non-annotation information as the file (if any) from which information was loaded to initialize the receiver with the following exceptions: - Annotations added via the receiver since information was loaded are saved in the specified file. - Annotations deleted via the receiver since information was loaded are not saved in the specified file. - Annotations whose attributes have been changed via the receiver since information was loaded are save with changed attributes in the specified file. Trace: REQ-041000, REQ-041050

Parameters
aPathA valid complete file system path including the name of the document to save. The path must be writable in a file system with sufficient available storage capacity to store the information encapsulated by the receiver.
Exceptions
java.io.IOException

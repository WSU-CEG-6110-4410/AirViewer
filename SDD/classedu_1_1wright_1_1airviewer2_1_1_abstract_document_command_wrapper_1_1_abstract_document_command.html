Detailed Description

This class defines the interface, variables, and common operations in order to encapsulates commands that may be executed following the Command Design Pattern.

Concrete subclasses must implements execute() When executed to perform operations specific to the concrete subclass and return a reciprocal command to be executed if/when the undo is required. If undo is inappropriate for some reason, concrete implementations of execute() may return null.
